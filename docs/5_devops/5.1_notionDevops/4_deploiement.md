# CI - Déploiement

![en connstruction](../../images/enConstruction.png)

Enfin une des tâches qui peut faire partie d’un pipeline est celle du déploiement d’une application sur un environnement de test, de développement ou de production. Cela permet de disposer en continue d’une plateforme à jour, disposant des dernières modifications si celles-ci ont passé toutes les étapes d’un pipeline sans renvoyer d’erreur.
